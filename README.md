# Calculator React App

A basic calculator web application built with React, featuring a clean and modern UI.

## Features

- Standard 4-function calculator (addition, subtraction, multiplication, division)
- 3x3 number pad layout
- Clean, modern styling with gradient background
- Responsive design
- Division by zero error handling
- Decimal point support

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm (comes with Node.js)

### Installation

1. Install dependencies:
```bash
npm install
```

### Running the App

Start the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173` (or the port shown in the terminal).

### Building for Production

Create a production build:
```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```
